# Sensor-de-temparatura-indoor
SENSOR DE TEMPERATURA CON LM35 + PIC18F4550 + LCD
Termómetro digital con alarma basado en el sensor LM35 y el microcontrolador PIC18F4550, que mide temperatura, con sistem de alertas visuales cuando se supera un umbral configurable.
Sistema de alarma con histéresis
La alarma se activa cuando la temperatura supera los 40 °C y no se apaga hasta que baja de 38 °C, evitando activaciones repetidas cuando la temperatura oscila cerca del umbral.
